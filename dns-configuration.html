Me First Group Website - DNS Configuration Guide

Important: Proper DNS configuration is critical for your website to be accessible at your domain (mefirstgroup.co.za).

DNS Records Overview

To ensure your website functions correctly, you need to set up the following DNS records:

Record Type Host/Name Value/Points To TTL Purpose
A @ [Your Server IP] 3600 Points your root domain to your hosting server
CNAME www @ 3600 Makes www.mefirstgroup.co.za point to the same place as mefirstgroup.co.za
MX @ Depends on email provider 3600 Required for email delivery
TXT @ v=spf1 [your spf record] 3600 Email authentication (SPF)

Step-by-Step DNS Configuration

  1. Log in to your domain registrar (e.g., GoDaddy, Namecheap, or where you purchased mefirstgroup.co.za)
  2. Navigate to the DNS management section (often called "DNS Settings," "DNS Management," or "Name Servers")
  3. Configure A Record
    • Type: A
    • Host/Name: @ (or leave blank, depending on the registrar)
    • Value/Points to: Your hosting server's IP address
    • TTL: 3600 (or 1 hour)
  4. Configure CNAME Record
    • Type: CNAME
    • Host/Name: www
    • Value/Points to: @ or your domain (mefirstgroup.co.za)
    • TTL: 3600 (or 1 hour)
  5. If using email with your domain, set up MX records according to your email provider's instructions
  6. Save changes and allow time for propagation (can take 24-48 hours, though often much faster)

DNS Propagation

After updating DNS records, changes can take time to propagate across the internet (up to 48 hours, though typically much faster).

You can check the current status of your DNS propagation using these tools:

  • WhatsmyDNS.net
  • DNSChecker.org
  • Dig Web Interface

Common DNS Issues and Solutions

Website Not Loading

  • Check A record: Verify that the A record points to the correct server IP address
  • Verify propagation: Use DNS checkers to see if changes have propagated
  • Check hosting server: Ensure your hosting server is running properly

www Subdomain Not Working

  • Check CNAME record: Ensure the www CNAME record is properly set up
  • Verify .htaccess: Make sure your .htaccess file properly handles www to non-www redirection (or vice versa)

Email Not Working

  • Check MX records: Verify that MX records are configured according to your email provider's instructions
  • Check SPF/DKIM: Ensure SPF and DKIM records are properly set up for email authentication
Need help? If you're having trouble with your DNS configuration, contact your domain registrar's support team or your web hosting provider for assistance.

Testing Your Website

After DNS propagation is complete, test your website by:

  1. Visiting both http://mefirstgroup.co.za and http://www.mefirstgroup.co.za
  2. Testing on different devices and browsers
  3. Checking that SSL (https://) is working properly if you've installed an SSL certificate
  4. Testing any email functionality if your site uses contact forms
