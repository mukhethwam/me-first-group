Me First Group Website - HTML Deployment Instructions

Note: This website has been specifically built to work with standard HTML hosting that doesn't support Node.js or server-side processing.

Building the Website for HTML Hosting

  1. Make sure you have Node.js installed on your development computer (only needed for building, not for hosting)
  2. Open a terminal in the project folder
  3. Run the build command: 
    npm run build
  4. This will create a dist folder with all the static HTML, CSS, and JavaScript files

Uploading to Standard HTML Web Hosting

  1. Upload all files from the dist folder to your web hosting service
  2. Use FTP, cPanel File Manager, or any upload tool provided by your hosting service
  3. Maintain the exact folder structure as it appears in the dist folder
  4. Make sure to upload all files, including:
    • index.html (main HTML file)
    • All folders (assets, etc.)
    • All JavaScript and CSS files
    • All images and other assets
Important! Do not rename any files or change the folder structure. All paths are relative and must be maintained exactly as they are.

Testing Your Website

After uploading all files:

  1. Navigate to your website's URL
  2. Verify that the homepage loads correctly
  3. Test navigation between all pages
  4. Test the contact form and other interactive elements
  5. Test on mobile devices to ensure responsive design works properly

Troubleshooting Common Issues

  • White screen or 404 errors: Ensure all files were uploaded correctly and the folder structure is maintained
  • Missing images: Verify that all image files were uploaded
  • Navigation issues: This site uses a hash router (#) for compatibility with static hosting. Ensure you didn't modify the URL structure

If you encounter any issues with your website after deployment, please contact your web development team for support.
