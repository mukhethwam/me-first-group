Me First Group Website - Extension-less Deployment Instructions

Important! This build has been configured to generate JavaScript files WITHOUT file extensions to work around issues with some hosting providers.

Uploading to Web Hosting

  1. Run the build script to generate the distribution files: 
    node build.js
  2. Upload all files from the dist folder to your web hosting, including files without extensions
  3. Make sure the .htaccess file is also uploaded (it may be hidden in your file browser)
  4. Upload to the root directory of your website (often public_html or www)

Server Requirements:

  • Apache web server with mod_rewrite enabled
  • Ability to use .htaccess files
  • Support for setting MIME types

Troubleshooting

  • Blank Page: Check that all files from the dist/assets directory were uploaded, including files with no extensions
  • 404 Error for JS files: Ensure your server is correctly processing the .htaccess file and has mod_mime and mod_rewrite enabled
  • Script not loading: Check browser console (F12) for specific error messages

Testing

After uploading:

  1. Clear your browser cache completely
  2. Visit your website URL
  3. If the page is still blank, open browser developer tools (F12) and check the Console tab for error messages
  4. Verify in the Network tab that all resources are loading correctly
