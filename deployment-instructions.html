Me First Group Website - Deployment Instructions

Building the Website

  1. Make sure all your changes are saved
  2. Run the build command: 
    npm run build
  3. This will create a dist folder with all the static files for your website

Hosting Options

Option 1: Standard Web Hosting

You can upload the contents of the dist folder to any standard web hosting service:

  1. Use FTP or the hosting provider's file manager to upload all files
  2. Make sure the index.html file is in the root directory
  3. Upload all folders (assets, images, etc.) to maintain the correct relative paths
  4. No server-side configuration is needed as this is a static site

Option 2: GitHub Pages

  1. Create a GitHub repository
  2. Push your code to the repository
  3. Enable GitHub Pages in the repository settings
  4. Set the source to the dist folder or main branch

Option 3: Netlify, Vercel, or similar

  1. Create an account on Netlify, Vercel, or similar service
  2. Connect your GitHub repository
  3. Configure the build settings:
    • Build command: npm run build
    • Publish directory: dist
  4. Deploy the site

Domain Configuration

Once your site is hosted, you can connect your custom domain (mefirstgroup.co.za):

  1. Purchase the domain if you haven't already
  2. In your domain registrar's settings, point the domain to your hosting provider
  3. If using a service like Netlify or Vercel, follow their instructions for adding a custom domain
  4. Configure SSL for secure HTTPS access

Post-Deployment Checklist

  • Verify all pages load correctly
  • Check that all links work properly
  • Test the website on different devices and browsers
  • Verify that the contact form works (if applicable)
